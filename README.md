## ProofNet
A decentralized anonymous network that uses bandwidth contribution proofs to achieve fairness while preserving privacy.

## Progress
- Basic source routing
- Nodes load keys at startup
- Next hop sends a signed receipt for bandwidth contributions
- Nodes verify and store receipts
- Onion routing
- Accounting nodes
